class  ca.axoninteractive.Collections.ConcurrentPriorityQueue< T >
 The ConcurrentPriorityQueue class is a thread-safe generic priority queue using a binary heap for sorting elements by priority. The underlying binary heap uses a min-heap property, therefore the element with the highest priority will always be dequeued first. If several items share the same priority, they are dequeued in the same order that they were enqueued (FIFO). More...
