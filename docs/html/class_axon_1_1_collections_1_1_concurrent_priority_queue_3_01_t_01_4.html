Detailed Description

The ConcurrentPriorityQueue class is a thread-safe generic priority queue using a binary heap for sorting elements by priority. The underlying binary heap uses a min-heap property, therefore the element with the highest priority will always be dequeued first.

Template Parameters
TThe type of data to be queued.
